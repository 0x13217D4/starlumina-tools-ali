# 星芒集盒

一个功能丰富的支付宝小程序，集成了多种实用工具和测试功能。

## 项目简介

星芒集盒是一个实用工具集合小程序，为用户提供便捷的日常工具和测试功能。当前版本为 1.0.5。

## 功能特性

### 🛠️ 实用工具
- **计算器** - 基础数学计算
- **单位转换器** - 各种单位换算
- **汇率转换器** - 货币汇率换算
- **条码生成器** - 生成各种条码
- **二维码生成器** - 生成二维码
- **扫码功能** - 扫描条码和二维码
- **随机数生成器** - 生成随机数
- **指南针** - 方向定位工具
- **传感器** - 设备传感器数据

### 📱 设备相关
- **设备信息** - 查看设备详细信息
- **屏幕测试** - 屏幕显示测试
- **多点触控** - 触摸屏测试
- **反应时间测试** - 测试反应速度
- **时间屏幕** - 时间显示功能

### 🧠 心理测试
- **MBTI测试** - 性格类型测试
- **BDI抑郁测试** - 抑郁自评量表
- **SAS焦虑测试** - 焦虑自评量表
- **SCL90症状测试** - 症状自评量表
- **SDS抑郁测试** - 抑郁自评量表

### 🎮 娱乐功能
- **24点游戏** - 数学益智游戏
- **2048游戏** - 数字合成游戏

### 📄 其他功能
- **图片压缩** - 图片大小压缩
- **身份证验证** - 身份信息验证
- **隐私政策** - 隐私保护说明

## 项目结构

```
星芒集盒_支付宝小程序/
├── pages/                  # 页面文件
│   ├── tools/             # 工具页面
│   ├── mine/              # 个人中心
│   ├── about/             # 关于页面
│   ├── calculator/        # 计算器
│   ├── converter/         # 转换器
│   ├── psychology-test/   # 心理测试
│   └── ...                # 其他功能页面
├── utils/                 # 工具函数
├── images/                # 图片资源
├── app.js                 # 小程序入口
├── app.json               # 小程序配置
├── app.wxss               # 全局样式
└── package.json           # 项目依赖
```

## 技术栈

- **前端框架**: 支付宝小程序原生开发
- **UI组件**: @miniprogram-component-plus
- **开发语言**: JavaScript
- **样式**: ACSS (支付宝小程序样式)

## 安装与运行

1. 克隆项目到本地
2. 使用支付宝小程序开发者工具打开项目
3. 配置小程序AppID
4. 点击编译运行
